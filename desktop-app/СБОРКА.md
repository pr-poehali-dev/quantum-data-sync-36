# Инструкция по сборке десктопного приложения

## Что нужно установить (один раз)

1. **Node.js** — https://nodejs.org (версия 20 LTS)
2. **Python 3.11** — https://www.python.org/downloads/release/python-3119/
   - При установке отметить галочку **"Add Python to PATH"**
3. **Git** (опционально) — https://git-scm.com

---

## Структура папок

```
desktop-app/
├── server/          ← Python Flask бэкенд
│   ├── app.py
│   ├── requirements.txt
│   └── data/        ← сюда создастся db.sqlite3 (база данных)
├── frontend/        ← исходники React-фронтенда (копия из основного проекта)
│   └── src/
│       └── lib/
│           └── api.ts  ← уже переключён на localhost:8787
├── electron/        ← Electron оболочка
│   ├── main.js
│   └── package.json
└── СБОРКА.md        ← эта инструкция
```

---

## Шаг 1 — Скачать код фронтенда

Скачай исходники проекта с poehali.dev:
**Скачать → Скачать код**

Распакуй в папку `desktop-app/frontend/` (чтобы там был `src/`, `package.json`, `vite.config.ts` и т.д.)

---

## Шаг 2 — Заменить api.ts

Замени файл `desktop-app/frontend/src/lib/api.ts` на файл из этой папки
(`desktop-app/frontend/src/lib/api.ts` — он уже подготовлен, запросы идут на localhost:8787).

---

## Шаг 3 — Собрать фронтенд

Открой командную строку (Win+R → cmd), перейди в папку frontend:

```cmd
cd desktop-app\frontend
npm install
npm run build
```

После этого появится папка `desktop-app/frontend/dist/` — это готовый фронтенд.

---

## Шаг 4 — Установить зависимости Python

```cmd
cd desktop-app\server
pip install -r requirements.txt
```

### Проверка сервера (опционально)

```cmd
python app.py
```

Должно появиться:
```
✓ База данных инициализирована
✓ Сервер запущен: http://localhost:8787
```

Нажми Ctrl+C чтобы остановить.

---

## Шаг 5 — Скачать встроенный Python (для .exe без зависимостей)

Чтобы на целевом ПК не нужен был установленный Python:

1. Скачай **Python embeddable package** (Windows 64-bit):
   https://www.python.org/ftp/python/3.11.9/python-3.11.9-embed-amd64.zip

2. Распакуй в папку `desktop-app/python-embedded/`

3. Установи pip в embedded Python:
   - Скачай https://bootstrap.pypa.io/get-pip.py
   - Выполни:
   ```cmd
   desktop-app\python-embedded\python.exe get-pip.py
   ```

4. Установи Flask в embedded Python:
   ```cmd
   desktop-app\python-embedded\python.exe -m pip install flask flask-cors
   ```

5. Отредактируй файл `python-embedded/python311._pth`:
   Раскомментируй строку `import site` (убери `#` в начале).

---

## Шаг 6 — Собрать .exe установщик

```cmd
cd desktop-app\electron
npm install
npm run build
```

Готовый установщик появится в:
```
desktop-app\electron\dist\Журнал успеваемости Setup 1.0.0.exe
```

---

## Запуск без сборки (для тестирования)

Можно запустить приложение без сборки в .exe:

**Терминал 1 — сервер:**
```cmd
cd desktop-app\server
python app.py
```

**Терминал 2 — фронтенд (режим разработки):**
```cmd
cd desktop-app\frontend
npm run dev
```

Открой браузер: http://localhost:5173

- **Пароль:** admin

---

## База данных

Файл базы данных: `server/data/db.sqlite3`

- Создаётся автоматически при первом запуске
- Для резервной копии — просто скопируй этот файл
- Для переноса данных на другой ПК — скопируй `db.sqlite3`
